Neural network

  • Each layer has nodes that contain activation function.
  • Initally, network makes random guess on answers. Then, it finds diff between actual and predicted, and make adjustment to weights via gradient descent.
import torch.nn as nn

## 20 is the hidden dimension.
model = nn.Sequential(
        nn.Linear(3,20),  # 3 is the input features x1,x2,x3.
        nn.ReLU(),
        nn.Linear(20,2))  # 22 is the classes.

print(model)

Set up

  • Input layer \( x = a^{[0]} \), hidden layer \( a^{[1]} \), Output layer \( \hat{y} = a^{[2]} \)
  • This is 2 layer network, because we do not count the input layer.

Single training example with 1 hidden layer

  • \( z^{[1]} = \textbf{w}^{[1]}\textbf{x} + \textbf{b}^{[1]} \)
  • \( a^{[1]} = \sigma{(z^{[1]})} \)
  • \( z^{[2]} = \textbf{w}^{[2]}a^{[1]} + \textbf{b}^{[2]} \)
  • \( a^{[2]} = \sigma{(z^{[2]})} \)

Activation

  • Non-linear functions that determine outputs of neurons.

Sigmoid

  • Normalizes the output between 0 and 1.
  • Function flattens at the edges where gradients are close to 0.

Tanh

  • Similar to sigmoid but ranges between 0 and 1.

Relu

  • Computationally efficient.
  • For all negative inputs, neuron will not activate, and the weights will not update.

Leaky Relu

  • Multiply all negative inputs with a small number.

Softmax

  • A different type of activation used for multi-class classification.
  • Assigns a probability to each class with all of them sum to 1.
