Vision transformers (ViT)

  • Split iamge into patches
    • Patches of equal dimensions represent words of sequence
    • Patches is to avoid processing images pixel by pixel
  • Linear projection of flattened images
    • Obtains a sequence of work-like patches
  • Hybrid input embedding sublayer
    • Add a convolutional layer to embed the linear projection of the patches
    • Add positional encoding to retain the structure of the original image
    • Then, process the embedded input with a standard original BERT-like encoder
