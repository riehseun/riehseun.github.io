Convolutional neural network

  • Has ability to extract and learn complex features.
  • Convolutional layer
    • In decision making, information needed is often spatially close.
    • We only need to take weighted sum of nearby inputs.
  • Ex. Inception v3, Xception, DenseNet, AlexNet, VGG16, ResNet
Reference: Deep Learning Specialization

Edge detection

  • Ex. 6x6 image * (convolve) 3x3 filter = 4x4 image
Reference: Deep Learning Specialization

Padding

  • Avoids shrinking output and throwing away information from edges.
  • "valid": no padding
  • "same": pad so that output size is the same as the input size
Reference: Deep Learning Specialization

Convolution over volumn

  • 6x6x3 volumn (height x width x number of channels) * 3x3x3 volume (height x width x number of channels) = 4x4 (27 numbers are multiplied and summed up 16 times to produce 4x4)
  • Number of channel must match between input and filter.
Reference: Deep Learning Specialization

Pooling layers

  • Learn low-level features like lines.
  • "max pooling": take max number in each region.
  • No parameters to learn.
  • 2 by 2 with stride 2 is commonly used.
Reference: Deep Learning Specialization
