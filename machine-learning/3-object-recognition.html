Object localization

  • Target label \( y \) is a vector whose elements contain
    • \( p_{c} \) - boolean indicating whether there is object or not
    • \( b_{x} \) - x coodinate
    • \( b_{y} \) - y coodinate
    • \( b_{w} \) - width
    • \( b_{h} \) - height
    • \( c_{1} \) - class label #1
    • \( c_{2} \) - class label #2
    • \( c_{3} \) - class label #3
  • Loss is \( (\hat{y}_{1} - y_{1})^{2} + \dots + (\hat{y}_{8} - y_{8})^{2} \)
  • If \( p_{c} = 0 \) we don't care about rest of the values. In this case, loss is just \( (\hat{y}_{1} - y_{1})^{2} \)

Object detection

Sliding window

  • Create boxes and move it around inside the image.
  • When image recognition algorithm detects the object, return.
  • When finished, change the size of boxes and move them around again.
  • Computation cost is very high.

Convolutional sliding window

  • Use convolution / maxpool / FC to compute all boxes in one shot rather than sliding the box around inside the image.
Reference: Deep Learning Specialization

Bounding box predictions

  • Split the image into smaller grid cells. For example \( 3 \times 3 = 9 \) grid cells.
  • For each grid cell, compute \( y \) vector.
  • Midpoint of each object is used to determine the grid that the object belongs.
Reference: Deep Learning Specialization

Intersection over union

  • Measure of the overlap between two bounding boxes.
  • Size of intersection / size of predicted bounding box.
  • Correct if IoU is greater than or equal to 0.5.
Reference: Deep Learning Specialization

Non-max suppression

  • Cleans up multiple detections of same object.
  • Discard all boxes when probability of whether there is an object is less than or equal to 0.6.
  • While there are any remaining boxes.
    • Pick the box with the largest probability and output that as a prediction.
    • Discard any remaining box with IoU greater than or equal to 0.5. with the box output in the previous step.
Reference: Deep Learning Specialization

Anchor boxes

  • Overlapping objects? Create multiple (anchor) boxes.
  • Each object in training image is assigned to grid cell that contains object's midpoint and anchor box for the grid cell with highest IoU.
