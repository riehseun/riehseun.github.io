Orthogonalization

  • Models are trained on train set.
    • Bigger network.
    • Better optimization algorithm.
  • Hyperpameters are selected on validation set.
    • Regularization.
    • Bigger training set.
    • As we do this, we are leaking information from validation set to training set. We may end up overfitting to validation data.
  • Final evaluation is done on test set.
    • Bigger dev set.
  • Perform in real world.
    • Change dev set.
    • Change cost function.

Solve linear regression

  • Matrix algebra
    • Inverse of matrix may not be computable.
    • Singular value decomposition could be used to compute the whole expression including the inverse.
  • Gradient descent
    • Take negative of gradients of a function at a point.
    • Repeatedly update that point until reaching an optimal point.
    • Batch: run through all samples to do single update of parameters.
    • Min-batch: run through subset of samples to do single update of parameters.
    • Stochastic: run through one sample to do single update of parameters.
