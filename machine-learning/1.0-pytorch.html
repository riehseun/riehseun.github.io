Pytorch basic

Tensor

  • Similar to array and matrix
  • Can run on GPU
  • Support automatic differentiation
  • Has tree attributes
    • shape
    • dtype
    • device

Dataset

  • Stores the samples and their corresponding labels
  • Custom dataset must implement three methods
    • __init__(self)
    • __len__(self)
    • __get_item__(self, index)

DataLoader

  • Wraps an iterable around Dataset
  • Use cases
    • Define mini batches
    • Reshuffle data at every epoch to reduce model overfitting
    • Use multiprocessing for faster data retrieval
# Tensor filled with zeros
torch.zeros(*size)

# Remove dimensions whose size is 1
torch.squeeze(x)

# Apply Relu element-wise
torch.relu(x)

# Return vector/matrix norm of given tensor
torch.norm(x)

# Softmax of tensor
torch.softmax(x)

# Softmax, then log the tensor
torch.log_softmax(x)

# Fills tensor with value where mask is True
Tensor.masked_fill_

  • Base class for neural network
    • import torch.nn as nn

ModuleList

  • List of submodules
    • linears = nn.ModuleList([nn.Linear(10, 10) for i in range(10)])

Embedding

  • Lookup table to store embedding
    • embedding = nn.Embedding(num_embeddings, embedding_dim)

Linear

  • Apply linear transformation
    • linear_1 = nn.Linear(in_features, out_features)

Dropout

  • During training, randomly zeros some elements of input tensor with probability p
    • dropout = nn.Dropout(p)

Parameter

  • Subclass of tensor
    • out = nn.Parameter(data)  # data should be tensor
