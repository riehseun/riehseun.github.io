Recurrent neural network

  • First need to extract numerical features from text data. (Ex. bag of words, N-grams, word embeddings)
  • Process the input timestep by timestep.
  • Backprop needs separate layer for each time step with the same weights for all layers.
  • Different loss per timestep.
  • Gradients from multiple timesteps are added to calculate the final gradient.

Why not standard network

  • Inputs, outputs can be of different lengths in different examples.
  • Does not share features learned across different positions of text.

RNN types

  • One to many (Ex. music generation)
  • Many to one (Ex. sentiment classification)
  • Many to many (Ex. named entity recognition)
  • Many to many (Ex. machine translation)

Dropout

  • In RNN, dropout is applied to input/output of each cell unit.
  • Dropout decreases the amount of cell computation, preventing overfitting the data.
