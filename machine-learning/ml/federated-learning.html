Historical method

Anonymization

  • Hide sensitive fields in dataset.
  • If same dataset where different fields are hidden exist, sensitive information could be retrived by joining the datasets together.

Restrict queries

  • Do not return results when queries look for very specific inforamtion.
  • Information could still be leaked when different queries (that return large results, thus don't belong to restriction category) are combined together.

Federated learning

  • Models are deployed to clients where data exist.
  • Only gradient updates (not data) are sent to centralized server.
  • The centralized server then averages the gradient udpates and apply to the central model.

Weakness

  • Each client could generate data differently. (Samples come from different distribution)
  • Models could recover training examples.
