Introduction to Agents for Structured Data

  • How to convert query to SQL
    • Query -> Input classifier -> (If DB look up needed) Semantic parser, to generate SQL query -> SQL compiler, to generate database result -> Response
  • How to handle SQL + free text
    • Option 1. classify query to decide to retrieve from DB or text
    • Option 2. retrieve answer from DB and text separately, then choose answer
      • We often need both to retrieve information to generate answers
    • Option 3. convert DB to text, then use text retrieval
      • We lose algebraic power of SQL DB
    • Option 4. retrieve answer from DB and text separately, then use LLM to combine answer
