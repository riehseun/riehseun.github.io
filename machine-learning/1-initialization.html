Zero initialization

  • If all weights are initalized to 0, all units compute the same function due to symmetry
    • Every neuron in each layer learns the same function
    • Effective the same thing as setting the number of neurons in eaach layer to 1
    • As a result, network is no better than logistic regression
    • Bias vectors can be initialized to zero
  • The weight matrices - \( W^{[1]}, W^{[2]}, W^{[3]} \dots W^{[L-1]}, W^{[L]} \)
  • The bias vectors - \( b^{[1]}, b^{[2]}, b^{[3]} \dots b^{[L-1]}, b^{[L]} \)
def initialize_parameters_zeros(layers_dims):

    parameters = {}
    L = len(layers_dims)

    for l in range(1, L):
        parameters['W' + str(l)] = np.zeros((layers_dims[l], layers_dims[l-1]))
        parameters['b' + str(l)] = np.zeros((layers_dims[l], 1))

    return parameters

Random initialization

  • Breaks symmetry by randomly initalizing weights
  • Poor initialization, for exampling randomly initializing weights to large numbers, leads to exploding/vanishing gradient
def initialize_parameters_random(layers_dims):

    np.random.seed(3)
    parameters = {}
    L = len(layers_dims)

    for l in range(1, L):
        parameters['W' + str(l)] = np.random.randn(layers_dims[l], layers_dims[l-1]) * 10
        parameters['b' + str(l)] = np.zeros((layers_dims[l], 1))

    return parameters

He initialization

  • Very similar to Xavier initialization
    • Xavier - sqrt(1/layers_dims[l-1])
    • He - sqrt(2/layers_dims[l-1])
  • Works well with ReLU activations
def initialize_parameters_he(layers_dims):

    np.random.seed(3)
    parameters = {}
    L = len(layers_dims) - 1

    for l in range(1, L + 1):
        parameters['W' + str(l)] = np.random.randn(layers_dims[l], layers_dims[l-1]) * np.sqrt(2 / layers_dims[l-1])
        parameters['b' + str(l)] = np.zeros((layers_dims[l], 1))

    return parameters
