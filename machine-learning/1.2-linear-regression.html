Linear regression

  • Fits line to data
  • Dependent or target variable y
  • Independent or predictor variable x
  • Assumes errors are independently and identically normally distributed
  • Single layer neural network with linear activation is identical with linear regression

Linear classifier

  • \( \textbf{y} = \textbf{W}\textbf{x} + \textbf{b} \)
    • \( \textbf{y} \) is output of the classifier
    • \( \textbf{x} \) is \( N \) dimensional vector
    • \( \textbf{W} \) is \( N \times N \) matrix
  • Data are provided in pairs \( (x,t) \)
    • \( x \) is input to the classifer
    • \( t \) is label

Linear classifier with 5 inputs and 10 outputs

import torch
import torch.nn as nn

classifier = nn.Linear(5, 10)

Mean squared error

  • \( \displaystyle\sum_{i=0}^n (y_{i} - t_{i})^2 \)
  • Goal is to find \( \textbf{W} \) that minimizes the cost
import torch
import torch.nn as nn

model = nn.Linear(10,3)
loss = nn.MSELoss()

## Use random input x.
input_vector = torch.randn(10)

## Assume classiciation with 3 clases.
target = torch.tensor([0,0,1])

pred = model(input_vector)
output = loss(pred, target)
print("Prediction: " ,pred)
print("Output: " , output)
