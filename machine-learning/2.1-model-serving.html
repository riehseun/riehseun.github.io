Detailed design

  • Horizontally scale API servers
  • Pre-pull latest models from model registry rather than pulling it during runtime
  • Before using the model to compute inference, check if user-inference data exists in pre-compute inference cache
  • When user features are not available at all (users not signed in, anonymous users, users not sharing geo-location) recommend popular ADs
