Hyperparameter tuning

  • Should use random sampling to choose the number of layers, number of features, etc.
  • Scale parameters accordingly such as log scale.
  • Panda: babysit one model.
  • Caviar: train many models in parallel.
  • Tuning methods

Hyperparameter

  • \( \alpha \) the learning rate is most important.
  • Mini-batch siez, the number of hidden units are second important.
  • The number of layers and learning rate decay are third important.
  • If using Adam, never tune \( \beta_{1}, \beta_{2}, \epsilon \)

Tune learning rate between \( 0.0001 \) and \( 1 \)

  • Need to sample from log scale, not linear scale.
