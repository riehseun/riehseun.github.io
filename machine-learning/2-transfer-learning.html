Tranfer learning

  • When transfer learning A to B
    • Task A and B have the same input.
    • Task A has a lot more data than task B.

Example

  • Use image recognition to do x-ray diagnosis.
  • Delete the last layer and weights that feed to the last layer.
  • Create new randomly initialized weights for the last layer.
  • Retain the network with x-ray dataset.
  • In this case, training the network for image recognition task is called pre-training
  • In this case, training the network for image x-ray diagnosis is called fine-tuning

Multitask learning

  • Cost function: \( J = \dfrac{1}{m}\displaystyle\sum_{i=1}^{m}\sum_{j=1}^{C} L(\hat{y}_{j}^{(i)},y_{j}^{(i)}) \)
    • \( C \) is the number of different tasks.
    • Unlike softmax regression, each data can have multiple labels.
  • When there is shared low-level features.
  • When amount of data for each task is similar.
  • Not used very much in practice.
