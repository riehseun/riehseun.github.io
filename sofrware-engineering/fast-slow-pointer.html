Happy Number

1. Slow pointer moves one step at a time whule fast pointer moves two steps at a time.

  • If converges, then the result fo slow pointer would be equal to 1.
  • If diverges, then the result of slow and fast pointer would be equal at one point.
  • Time: O(mlogn) log(n) for the recursion and m is the number of digits in the input.
  • Space O(1).
def is_happy_number(n):

    return run(n, n)

def run(slow, fast):

    slow = get_squared_sum_of_digits(slow)
    fast = get_squared_sum_of_digits(get_squared_sum_of_digits(fast))

    # Converges to 1.
    if slow == 1:
        return True

    # Cycle detected. (Endless loop)
    if slow == fast:
        return False

    return run(slow, fast)

def get_squared_sum_of_digits(n):

    ssd = 0

    for i in str(n):
        ssd = (int(i)**2)

    return ssd
