Happy Number

1. Slow pointer moves one step at a time whule fast pointer moves two steps at a time.

  • If converges, then the result fo slow pointer would be equal to 1.
  • If diverges, then the result of slow and fast pointer would be equal at one point.
  • Time: O(mlogn) log(n) for the recursion and m is the number of digits in the input.
  • Space O(1).
def is_happy_number(n):

    return run(n, n)

def run(slow, fast):

    slow = get_squared_sum_of_digits(slow)
    fast = get_squared_sum_of_digits(get_squared_sum_of_digits(fast))

    # Converges to 1.
    if slow == 1:
        return True

    # Cycle detected. (Endless loop)
    if slow == fast:
        return False

    return run(slow, fast)

def get_squared_sum_of_digits(n):

    ssd = 0

    for i in str(n):
        ssd = (int(i)**2)

    return ssd

Linked list cycle

1. Move slow and faster pointer to see if they meet.

  • If fast pointer exits linked list, return False.
  • If two pointers are the same at any point, return True.
  • Time: O(n).
  • Space: O(1).
def detect_cycle(head):

   return run(head, head)

def run(slow, fast):

   slow = slow.next

   if not fast or not fast.next:
      return False

   fast = fast.next.next

   if slow == fast:
      return True

   return run(slow, fast)

Circular array loop

1. Move slow and faster pointer to see if they meet.

  • If either of slow or fast pointer changes direction, return False for the particular starting index.
  • If two pointers are the same at any point, return True for the particular starting index.
  • If a value at a particular index equals to the length of array, return False.
  • Time: O(n^2).
  • Space: O(1).
def circular_array_loop(arr):

    result = False

    # If find one index i at which there is a cycle, must return True.
    for i in range(len(arr)):
        result_i = run(i, "", i, "", arr)
        result = result or result_i

    return result

def run(slow, slow_direction, fast, fast_direction, arr):

    slow, slow_direction = move_index(slow, slow_direction, arr)
    fast, fast_direction = move_index(fast, fast_direction, arr)
    fast, fast_direction = move_index(fast, fast_direction, arr)

    # Direction changes in either slow or fast pointer, not a cycle.
    if slow == -1 or fast == -1:
        return False

    # If two pointers match at one point, cycle.
    if slow == fast:
        return True

    return run(slow, slow_direction, fast, fast_direction, arr)

def move_index(i, current_direction, arr):
    index = i + arr[i]

    # Loop with just one element, not a cycle.
    if abs(arr[i]) % len(arr) == 0:
        return -1, ""

    if arr[i] > 0:
        if current_direction == "neg":
            return -1, ""
        current_direction = "pos"
    elif arr[i] < 0:
        if current_direction == "pos":
            return -1, ""
        current_direction = "neg"

    if index >= len(arr):
        index = index % len(arr)

    if index < 0:
        index = len(arr) - (abs(index)%len(arr))

    return index, current_direction

Middle of linked list

1. Move slow and fast pointer from the head of the linked list.

  • If fast pointer exits linked list, slow pointer would be located in the middle of the linked list.
  • From the middle pointer, reverse second half of the linked list.
  • Traverse from both ends of the linked list towards the middle.
  • Time: O(n).
  • Space: O(1).
def palindrome(head):

    start = head
    middle = run(head, head)
    middle_save = middle

    # Reverse the second half of linked list
    prev = None
    nex = None
    while middle:
        nex = middle.next
        middle.next = prev
        prev = middle
        middle = nex
    end = prev

    # Iterate linked list from both ends simultaneously.
    while True:

        print(start.data)
        print(end.data)

        # If odd, both pointers will meet at the middle point.
        if start == end:
            return True

        # If even, this condition will be met around the middle point.
        if start.next == end:
            if start.data != end.data:
                return False
            else:
                return True

        # If at any point difference is found.
        if start.data != end.data:
            return False

        start = start.next
        end = end.next

    return True
