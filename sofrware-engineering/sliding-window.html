Find maximum in sliding window

1. Iterate each window and compute the max

  • Time: O(nw) where n is list length and w is size of window.
  • Space: O(1) there is no extra space used during computation.
def find_max_sliding_window(nums, window_size):

    result = []
    start = 0
    end = window_size

    while end <= len(nums):
        result.append(max(nums[start:end]))
        start +=1
        end += 1

    return result

2. Use a queue to keep track of the maximum

  • Keep the maximum number at the front of the queue. (We can do this by popping out all smaller numbers than the current number)
  • First, compute the maximum number in the first window.
  • Then, everytime a window moves
    • Pop out all smaller numbers than the current number.
    • Remove previous number, which is now outside of the window, from the queue.
    • Append the first item in the queue to the result.
  • Time: O(n) where n is list length.
  • Space: O(w) where w is the window size.
from collections import deque

def find_max_sliding_window(nums, window_size):
    result = []
    queue = deque()

    # Compute the maximum in the first window.
    for i in range(window_size):

        # Pop out all smaller items than the current item
        # by removing last items in the queue.
        # Then the first itme in the queue will always
        # contains the maximum item.
        while queue and nums[queue[-1]] < nums[i]:
            queue.pop()
        queue.append(i)

    result.append(nums[queue[0]])

    # Move the window
    for i in range(window_size, len(nums)):

        while queue and nums[queue[-1]] < nums[i]:
            queue.pop()
        queue.append(i)

        # Remove all indexes in the queue that is before
        # and outside than the current window.
        while queue and queue[0] <= (i - window_size):
            queue.popleft()

        result.append(nums[queue[0]])

    return result

  • Window size is greater than the list length. ([1,2], 3)
  • Window size is zero. ([1,2], 0)
  • Window size and list length are zero. ([], 0)

Minimum window subsequence

1. Use hash table to store indexes where each character appears.

  • Time: O(n^2) where n is length of str1.
  • Space: O(n) where n is length of str1.
import math

def min_window(str1, str2):
    char = {}

    for i in range(len(str1)):
        if str1[i] not in char:
            char[str1[i]] = []
        char[str1[i]].append(i)

    candidate_strings = []
    first_match = str2[0]
    all_other_matches = str2[1:]

    if first_match not in char:
        return ""

    # Must interate all indices of first chracter match.
    # Construct shortest valid string starting at each first character match.
    for index in char[first_match]:
        index_array = []
        index_array.append(index)

        # Most possible number of checks will not exceed the number
        # of characters in str1.
        for c in all_other_matches:
            if c not in char:
                break

            for index in char[c]:
                # Append the smallest index possible.
                if index > index_array[-1]:
                    index_array.append(index)
                    break

        if len(index_array) >= len(str2):
            candidate_strings.append(str1[index_array[0]:index_array[-1]+1])

    min_length = math.inf
    min_string = ""
    for string in candidate_strings:
        if len(string) < min_length:
            min_length = len(string)
            min_string = string

    return min_string

2. Use two pointers.

  • Increment index of str1 until all characters in str2 is found in str1.
  • Time: O(n) where n is length of str1.
  • Space: O(1) no extra space needed.
import math

def min_window(str1, str2):

    result = ""
    min_distance = len(str1) + 1

    index1 = 0
    index2 = 0

    while index1 < len(str1):

        if str1[index1] == str2[index2]:
            index2 += 1

            # At this point, str1 contains all chracters of str2.
            # index1 is at the last matching character.
            # index2 is at the last matching character plus 1.
            if index2 == len(str2):
                # Bring the index2 to the last index of str2.
                index2 -= 1
                # Idea: the last chracter in this range is guaranteeed to
                #       be the part of the solution in this range.

                start = index1
                end = index1

                while index2 >= 0:
                    if str1[start] == str2[index2]:
                        index2 -= 1
                    start -= 1

                # When index2 is less than 0, then we covered all characters in
                # str2.
                # Reset the position of index2 to 0. (At this point, it would be -1)
                index2 = 0

                if end - start < min_distance:
                    min_distance = end - start
                    result = str1[start+1:end+1]

        index1 += 1

    return result

  • There is no solution ("beta", "ab")
  • length of str2 is greater than length of str1 ("beta", "betac")
  • str1 is empty ("", "abc")
  • str2 is empty ("abc", "")

Repeated DNA subsequence

1. Use hash table to store occurances of all substrings with length k.

  • Time: O(k * len(s)).
  • Space: O(len(s)-k) where number of substring to store is len(s)-k.
def find_repeated_sequences(s, k):

    substrings = set()
    result = set()

    for i in range(0, len(s)-k):
        substring = s[i:i+k]
        # To save set space, use hash of substring
        # Instead of the actual substring.
        if hash(substring) not in substrings:
            substrings.add(hash(substring))
        # Substring is repeated.
        else:
            result.add(substring)

    return result

  • Either one of k or len(s) is 0.

Mimimum window substring

1. Use two pointers and two hash tables.

  • Prepare start and end pointers and two hash tables for s and t.
  • Compute occurance of each character in t and store it in a hash table.
  • Iterate s by incrementing the end pointer until the substring contains all chars in t. Remember this substring.
  • This substring may not be the "minimum" window substring. Now increment the start pointer.
    • If the substring contains all chars in t, keep incrementing the start pointer. (as long as start <= end)
    • Otherwise, increment the end pointer.
  • Time: O(len(s)).
  • Space: O(len(t)).
def min_window(s, t):

    # In count_s, we will store the count of only the characters
    # that exist in t.
    count_s = {}
    count_t = {}

    min_length = len(s) + 1
    min_substring = ""

    for c in t:
        if c not in count_t:
            count_t[c] = 0
        count_t[c] += 1

    start = 0
    end = 0

    while start <= end and end <= len(s):

        if count_s.keys() == count_t.keys():

            if end - start < min_length:
                min_length = end - start
                min_substring = s[start:end]

            # Whenever "start" pointer moves forward, must remove
            # the previous chracter from count_s set.
            # This block must come before incrementing the start pointer!
            if s[start] in count_t:
                if count_s[s[start]] > 1:
                    count_s[s[start]] -= 1
                else:
                    del count_s[s[start]]

            start += 1

        else:
            # If covered the last character in s,
            # increment the start pointer until it reaches the end.
            if end == len(s):
                start += 1
            else:
                # Update count_s dictionary each time "end" pointer moves.
                # This block must come before incrementing the end pointer!
                if s[end] in count_t:
                    if s[end] not in count_s:
                        count_s[s[end]] = 0
                    count_s[s[end]] += 1

                end += 1

    return min_substring

Test cases

  • Either len(s) or len(t) is 0. ("" , "ABC"), ("ABCD" , "")
  • The number of unique chars in t is greater than the number of unique chars in s. ("ABCDDA" , "ABCEF")

Longest substring without repeating characters

1. Use two pointers and a hash table.

  • Iterate the string by moving end pointer from the beginning to the end.
  • Start pointer is placed at an index for each substring without repeating characters.
  • A hash table stores each character and its last occuring index.
  • If duplicate chacter is found, move the start pointer, compute the substring length, and update the hash map.
  • Else, just update the hash map with the new character.
  • Time: O(len(input_string)).
  • Space: O(len(input_string)).
def find_longest_substring(input_string):

   start = 0
   max_length = 0
   # Store chracter and its index.
   char_map = {}

   for i in range(len(input_string)):

      # If duplicate is found, compute the length using start and the i pointer.
      # The second condition ensures that we are only looking at the substring
      # after the start index.
      if input_string[i] in char_map and char_map[input_string[i]] >= start:
         max_length = max(max_length, i-start)
         # Move the start pointer to plus 1 of the index where the duplicate is.
         start = char_map[input_string[i]] + 1
      char_map[input_string[i]] = i

   max_length = max(max_length, len(input_string)-start)

   return max_length

Test cases

  • Entire string consists of a single char. ("aaaaaaaaaaaaaaaaa")
  • String is empty. ("")
  • Repeat of the same substring. ("ababababab")

Minimum size subarray sum

1. Use two pointers.

  • Iterate the array with start and end pointers.
  • Move the end pointer until the sum is >= target. Store the legnth of subarray.
  • Then, move the start pointer until the sum is < target.
  • Continue until both start and end finish.
  • Time O(len(nums)).
  • Space O(1).
def min_sub_array_len(target, nums):

    if sum(nums) < target:
        return 0

    min_length = len(nums)

    start = 0
    end = 0

    # Move end pointer until the sum is >= target
    # Then, move start pointer until the sum is < target
    # Continue until both start and end finish

    while start <= end and end < len(nums):

        if sum(nums[start:end+1]) >= target:
            min_length = min(min_length, end-start+1)
            start += 1
        else:
            # If end pointer reached the end of string.
            if end == len(nums) - 1:
                start += 1
            else:
                end += 1

    return min_length

Test cases

  • Empty array. (12, [])
  • Single element. (0, [1])
  • Impossible case. (9, [2,2])

Best Time to Buy and Sell Stock

1. Iterate once. Remember the minimum price and maximum profit seen so far.

  • Always remember the minimum price and maximum profit seen so far.
  • While iterating the array, the value is either lower or higher than the minimum seen so far.
    • If lower than the minimum, update the minimum seen so far.
    • If greater than the minumum, update the maximum profit by (price - minimum seen so far)
  • Time O(len(stock_prices)).
  • Space O(1).
def max_profit(stock_prices):

    min_price = max(stock_prices)
    max_profit = 0

    for price in stock_prices:

        if price < min_price:
            min_price = price
        else:
            max_profit = max(max_profit, price-min_price)

    return max_profit
