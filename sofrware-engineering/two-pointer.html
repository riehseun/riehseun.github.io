Valid palindrome

1. Compare chars at each end and move pointers towards the middle.

  • For the string with odd length, remove the middle char to simplify computation.
  • Time: O(n) where n is length of s.
  • Space: O(1).
def is_palindrome(s):

    if len(s) % 2 != 0:
        s = s[0:len(s)//2] + s[len(s)//2+1:]

    start = 0
    end = len(s) - 1

    while start < end:
        if s[start] != s[end]:
            return False
        start += 1
        end -= 1

    return True

Test cases

  • Empty string. ("")
  • String with single char ("a")

Three sum

1. Sort the array. Fix a number and move two pointers around.

  • Sort the array.
  • Iterate the array.
  • For each number, set low as numbers less than the number and set high as numbers greater than the number.
  • Check different values of low and high to see if target can be met.
  • Time: O(n^2) where n is length of array.
  • Space: O(1).
def find_sum_of_three(nums, target):

   nums.sort()

   for i in range(len(nums)):
      low = 0
      high = len(nums) - 1

      while low < i and i < high:
         print(str(nums[i]) + str(nums[low]) + str(nums[high]))
         if nums[i] + nums[low] + nums[high] == target:
            return True
         elif nums[i] + nums[low] + nums[high] < target:
            low += 1
         else:
            high -= 1

   return False

Test cases

  • Number of element is less than 3 ([1,2])

Reverse words

1. Swap each words.

  • Preprocess the sentence to remove extra whitespaces.
  • Convert the sentence into an array of words.
  • Swap words at each end while moving start and end pointer towards the middle.
  • Time: O(n) where n is length of sentence.
  • Space: O(1).
def reverse_words(sentence):

   # Convert multiple whitespaces to a single whitespace.
   # Split O(n), join O(n)
   sentence = " ".join(sentence.split())

   sentence_array = sentence.split(" ")

   start = 0
   end = len(sentence_array) - 1

   while start < end:
      temp = sentence_array[start]
      sentence_array[start] = sentence_array[end]
      sentence_array[end] = temp
      start += 1
      end -= 1

   return " ".join(sentence_array)

Test cases

  • Empty string ("")

Valid palindrome 2

1. Compare chars at each end and move pointers towards the middle.

  • Allow 1 mismatch to happen. If more than 1 mismatch, return False.
  • When mismatch happens, either increment start pointer or decrement end pointer such that two chars at the end matches. If cannot find match, return False.
  • Time: O(n).
  • Space: O(1).
def is_palindrome(s):

    start = 0
    end = len(s) - 1
    num_mismatch = 0

    while start < end:
        if s[start] == s[end]:
            start += 1
            end -= 1
        else:
            if num_mismatch == 0:
                num_mismatch += 1
                if s[start+1] == s[end]:
                    start += 1
                elif s[start] == s[end-1]:
                    end -= 1
                else:
                    return False
            else:
                return False

    return True

Test cases

  • Empty string. ("")
  • String with single char ("a")
  • String with two chars ("ab")
