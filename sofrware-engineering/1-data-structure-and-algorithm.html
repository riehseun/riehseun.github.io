Big O

Nested for-loop with dependant variables: O(n^2)

for i in range(n):
    for x in range(i):
        print(x)

Nested for-loop with index modification: O(n^2)

for i in range(n):
    i *= 2
    for x in range(i):
        print(x)

Loop statement that multiplies/divides the loop variable: O(log(n)/log(k))

while i < n:
    i *= k
    print(i)

Linked list

  • Access: O(n)
  • Insert (at head): O(1)
  • Delete (at head): O(1)
  • Insert (at tail): O(n)
  • Delete (at tail): O(n)
# Singly.
class Node:
    def __init__(self, data=0, next=None):
        self.data = data
        self.next = next

# Doubly.
class Node:
    def __init__(self, data=0, next=None, prev=None):
        self.data = data
        self.prev = prev
        self.next = next

Stack

# Initialize a stack.
stack = []

# Adding elements at the end of a stack.
stack.append('element')

# Removing elements from the end of a stack.
stack.pop()

# Get the item at the front of a stack.
stack[0]

# Get the item at the end of a stack.
stack[-1]
