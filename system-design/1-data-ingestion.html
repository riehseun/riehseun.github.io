# Requirement and Problem

What data sources are there? NAS drive, API, DB, etc
How is data produced? NAS - file drop, API - push and pull, DB - change data capture
What is size of data ingested per day? 10GB per day
What is growth rate of data? Assume 10x in 2 years =? 100GB per day
What is peak load? If evenly distributed, 10GB/day = 100KB/s, Assuming 20x peak, 2MB/s 
Is the data generated in batch or real-time? Both
What is latency requirement for real-time data to become available for consumption? under one second
How is data used? build ML models
How long should data be retained? 5 years
What are formats of data? csv, text file, parquet, JSON, XML, etc
In which format should the system store data? parquet
For real-time data, can Kafka be used? Yes
For processing batch data, can Spark be used? Yes
Should the system guarantee exactly-once or at-least-once delivery? At-least-once is needed
